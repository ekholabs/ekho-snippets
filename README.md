# EKHO Snippets

A collection of useful code snippets, tools, and utilities for the EKHO ecosystem.

## Available Snippets

### [Obsidian Link Converter](obsidian-link.html)
Convert Obsidian URI links (`obsidian://open?vault=...`) into clickable HTML links.

## Usage

All snippets are standalone HTML files that work in any modern browser. Simply:

1. Open the HTML file in your browser
2. Use the tool directly
3. Or download and integrate into your own projects
